# Secure User Identity Management System (RBAC)

A robust, enterprise-ready authentication system built with **Node.js**, **Express.js**, and **MongoDB**. This project implements secure user lifecycle management, including sign-up, login, and session handling using **JWT (JSON Web Tokens)** and **Bcrypt** hashing.

* **Secure Authentication:** Complete user sign-up, login, and logout flows.
* **Token-Based Security:** Implements **JWT** to manage stateless user sessions and protect private routes.
* **Data Security:** Utilizes **Bcrypt** for industry-standard password hashing and salting.
* **Database Management:** Optimized schemas using **MongoDB** and **Mongoose** for reliable data storage.
* **Input Validation:** Robust email and data validation to prevent malformed requests.
* **Session Handling:** Middleware integration (cookie-parser) for secure token management.
